# Deep Dive — Apuntes y ejemplos

Este repositorio contiene apuntes y ejemplos organizados por secciones para estudiar conceptos básicos y avanzados de Python.

## Estructura del proyecto

- `Seccion01/` — Conceptos fundamentales: tipos, estructuras de datos, control de flujo, funciones, excepciones y clases.

Cada archivo dentro de `Seccion01` contiene explicación y ejemplos ejecutables. Para ver el detalle de esa sección, consulta:

- [Seccion01/README.md](Seccion01/README.md)

## Cómo ejecutar los ejemplos

1. Abre una terminal en la raíz del proyecto (`Deep_Dive`).
2. (Opcional) activa tu entorno virtual si tienes uno.
3. Ejecuta cualquier archivo de ejemplo: por ejemplo

```bash
python -m Seccion01.Class01_hierarchy
python -m Seccion01.class10_part2
```

> Nota: los nombres de módulo respetan mayúsculas/minúsculas; en Windows la importación puede ser tolerante, pero es buena práctica mantener coincidencia exacta.

## Siguientes pasos

- Ejecuta los scripts en `Seccion01` y modifica los ejemplos para practicar.
- Puedo añadir un `requirements.txt`, instrucciones para crear un entorno virtual o notebooks interactivos si quieres.
